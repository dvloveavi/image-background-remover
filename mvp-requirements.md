## 2. 技术架构

### 2.1 技术栈

| 层级 | 技术方案 |
|------|----------|
| 前端托管 | Cloudflare Pages（免费） |
| API层 | Cloudflare Worker |
| 图片处理 | Remove.bg API ($0.12/张) |
| 域名解析 | Cloudflare DNS + CDN |
| 图床/存储 | 无（内存直接处理） |

### 2.2 数据流

```
用户选择图片
    ↓
浏览器预览（本地）
    ↓ POST (base64)
Cloudflare Worker
    ↓
Remove.bg API
    ↓
返回透明PNG
    ↓
浏览器展示 + 下载
```

## 3. 功能需求

### 3.1 核心功能

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 图片上传 | P0 | 支持拖拽、点击上传，限制jpg/png/webp，≤10MB |
| 实时预览 | P0 | 上传后本地预览原图 |
| 背景移除 | P0 | 一键处理，显示处理进度 |
| 结果下载 | P0 | 下载透明PNG |
| 历史记录 | P1 | 最近5次（仅浏览器本地） |

### 3.2 交互流程

```
┌─────────────┐
│  上传图片   │
└──────┬──────┘
       ↓
┌─────────────┐
│  本地预览   │ ← 用户选择文件后立即显示
└──────┬──────┘
       ↓
┌─────────────┐
│  点击移除   │ → 显示 loading spinner
└──────┬──────┘
       ↓
┌─────────────┐
│  展示结果   │ → 透明背景预览
└──────┬──────┘
       ↓
┌─────────────┐
│  下载按钮   │ → 触发浏览器下载
└─────────────┘
```

## 4. 非功能需求

### 4.1 性能

| 指标 | 目标 |
|------|------|
| 首屏加载 | < 2s |
| 图片处理 | < 5s（取决于Remove.bg响应速度） |
| 可用性 | 99.9% |

| 项目 | 限制 |

## 6. 竞品分析

| 竞品 | 优点 | 缺点 |
|------|------|------|
| remove.bg | 精度高、品牌强 | 付费较贵 |
| Clipdrop | 免费、有生态 | 需要账号 |
| Canva | 功能多、流量大 | 背景移除入口深 |

**差异化方向：** 极简、专注、速度快、免费额度高
